"ADVANCED" AI TECHNIQUES APPLIED INTO A GAME

When I entered the university, the first year, one subject gave us the option to program a "simple" minesweeper in Python as a voluntary exercise, and I took it as a challenge because I knew really litte about programming, but it came out great in the end.

The game is really easy to play and self explanatory, first you introduce a dificulty level and each turn you decide an action to take using a number from 1 to 5.
