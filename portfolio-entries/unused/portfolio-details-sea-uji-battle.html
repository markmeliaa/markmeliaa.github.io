A CLASSIC BATTLESHIP GAME

This project was born from a University task that I get really interested into, because it was an opportunity not only to introduce AI in a (basic) game but also to learn more about AI algorithms and how to make the CPU act smart.

The way to play it is really easy, in the main menu you select if you want sound effects and if you want to face an smart enemy or a simple enemy (CPU will perform random shots), then you place the boats wherever you want and you start playing the game against the AI.
