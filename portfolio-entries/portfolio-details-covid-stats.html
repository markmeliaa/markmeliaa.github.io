DAILY STATS FOR COVID-19

A mobile app for Android users that first takes the info requested to the user, you will have to enter at least a proper country and a start date and an end date in a proper interval of time (you can also introduce regions and subregions if the country you introduced has any) and once you have introduced those, you can start the search.

This app made in Kotlin (you can try it in Android Studio) that uses an API from the internet, reads it and interprets its data, and then displays the results of each requested day in cronological order, and it not only shows each day's cases, but also its deaths and recovered cases too.
